# Troubleshooting E: 일반형 (프로젝트 수행/성과 달성)

## Template
상황 → 취한 행동들 (action_1~5) → 최종 결과 → 인사이트

**E형 적용 케이스:**
- 문제 해결보다는 **프로젝트 전체 수행** 또는 **성과 달성**을 보여주는 경우
- 시도→개선 흐름이 아닌 **순차적 행동**을 나열하는 경우
- 예: 배포 자동화 구축, 팀 프로젝트 수상, 서비스 런칭 등

**주석 사용 가이드:**
- action은 1~5개까지 가변적으로 사용 (필요한 만큼만)
- 각 action은 title + desc로 구성
- SECTION 구조는 선택적 사용

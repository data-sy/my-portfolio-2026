## 7. 특이사항

### 워터마크 스타일
- 프로젝트명: 좌측 정렬, 48px Bold
- 버전: 프로젝트명 우측에 간격 12px, baseline 정렬, 18px Regular
- 전체 가로 배치 (HORIZONTAL)

### Attempt Card 구조
```
┌─────────────────────────────────────┐
│ ■  ┌──────┐                         │
│ ■  │시도 1│  쿼리 최적화             │  ← 배지 + 제목
│ ■  └──────┘                         │
│ ■                                   │
│ ■  N+1 문제 해결을 위해 fetch join 적용│  ← 설명
│ ■                                   │
│ ■  → 결과: 2.3s → 1.8s (22% 개선)    │  ← 결과 (초록색)
│                                     │
└─────────────────────────────────────┘
  ↑ 좌측 강조바 (4px, accentBlue)
```

### Conclusion Box 구조
```
┌─────────────────────────────────────┐
│  ✅ 결과                            │  ← 라벨 (14px, accentGreen)
│                                     │
│  2.3s → 0.4s  (85%↓)               │  ← 숫자(32px) + 퍼센트(24px, green)
│                                     │
└─────────────────────────────────────┘
```

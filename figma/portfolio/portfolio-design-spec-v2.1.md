## 4. Project Intro 레이아웃 (v2.1 재정의)

### 전체 구조
```
세로 비율: 이미지 영역 (1) : 텍스트 영역 (2)
가로 비율 (텍스트 영역 내): 좌측 (1) : 우측 (2)
```

### Type A: 성과 중심 (Traffic, MMT, Skeleton, Plogging)
- 이미지 영역: **Metric Hero** (성과 지표 3개)
- 좌측: 기간, 역할, 기술스택, **프로젝트 개요** 🆕
- 우측: 한 일, 정량적 성과, 인사이트

### Type B: 이미지 중심 (QuickLabelTimer)
- 이미지 영역: **앱 스크린샷/목업**
- 좌측: 기간, 역할, 기술스택, **프로젝트 개요** 🆕
- 우측: 한 일, 인사이트

### 레이아웃 비율 계산
```
콘텐츠 영역: 714 × 1027px (마진 제외)
워터마크 높이: ~60px
남은 높이: ~967px

이미지 영역: 967 × 1/3 ≈ 322px
텍스트 영역: 967 × 2/3 ≈ 645px

텍스트 영역 내:
- 좌측 (1/3): 714 × 1/3 ≈ 238px
- 우측 (2/3): 714 × 2/3 ≈ 476px
- 간격: 24px 고려하면 → 좌측 220px, 우측 470px
```

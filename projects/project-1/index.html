프로젝트 개요

데이터 라벨링 작업자를 위한 iOS 타이머 애플리케이션입니다. 작업 시간을 정확하게 측정하고 생산성을 향상시킵니다.

직관적인 UI와 백그라운드 타이머, 로컬 알림 기능을 제공합니다.

기술 스택

  • Swift 5.0
  • UIKit
  • CoreData (로컬 저장)
  • UserNotifications
  • Timer & RunLoop

기간

2025.10 - 2025.12 (3개월)

역할

개인 프로젝트 - 기획, 설계, 개발, 배포

주요 성과

  • App Store 출시: TestFlight 베타 테스트 후 정식 배포 완료
  • 사용자 피드백 반영: 5회 이상 업데이트를 통한 기능 개선
  • 백그라운드 안정성: 앱 종료 후에도 타이머 정확도 99.8% 유지
  • 데이터 손실 0%: CoreData 활용으로 로컬 데이터 안전하게 보존

주요 기능

  • 정밀 타이머: 시작/일시정지/재시작 기능으로 작업 시간 정확하게 측정
  • 백그라운드 동작: 앱이 백그라운드에서도 타이머 계속 작동
  • 로컬 알림: 설정한 시간이 되면 푸시 알림으로 알려줌
  • 작업 기록 저장: CoreData로 과거 작업 기록 조회 가능
  • 통계 대시보드: 일/주/월 단위 작업 시간 통계 제공

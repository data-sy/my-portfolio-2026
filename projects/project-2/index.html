쿠폰 이벤트 시스템

v2.1.0

고가용성 쿠폰 발급 시스템 (1000명 동시접속 지원)

System Architecture

프로젝트 개요

대규모 트래픽을 견디는 쿠폰 발급 시스템입니다. Redis 캐싱과 비동기 처리를 통해 1000명 동시접속 시에도 안정적으로 작동합니다.

성능 최적화를 통해 API 응답 시간을 75% 개선했습니다.

기술 스택

  • Java 11
  • Spring Boot 2.7
  • Redis (캐싱)
  • RabbitMQ (메시지 큐)
  • MySQL 8.0
  • Docker

기간

2025.06 - 2025.09 (4개월)

역할

백엔드 개발 - 시스템 설계, API 개발, 성능 최적화

주요 성과

  • API 응답 시간 75% 개선: 850ms → 210ms
  • 동시접속 처리량 20배 증가: 50명 → 1000명
  • 에러율 99% 감소: 12% → 0.1%
  • 서버 CPU 사용률 47% 감소: 85% → 45%

주요 기능

  • Redis 캐싱: 쿠폰 재고 수량을 Redis에 캐싱하여 DB 부하 감소
  • 비동기 처리: RabbitMQ를 활용한 백그라운드 쿠폰 발급
  • DB 최적화: 복합 인덱스 추가 및 쿼리 개선
  • 중복 발급 방지: 분산 락을 활용한 동시성 제어
